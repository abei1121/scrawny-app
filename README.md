# Scrawny (你是细狗吗) - 硬核训练追踪与 Web3 变现社区

**[点击这里直接访问 Scrawny 官方应用](https://raw.xiaojiucai.pro)**

## 1. 产品核心定位 (Core Identity)
Scrawny 是一款跨越门槛的硬核健身追踪平台，完美适配从零基础小白到专业健身教练。系统采用 100% 纯净无广告模式，为全球 8 大语种用户提供训练记录、视频教学、链上存证与流量变现的基础设施。

## 2. 核心功能架构 (Core Features)
* **智能外链教学 (Automated Video Tutorial Routing):** 点击动作即可自动跳转抖音、YouTube、Instagram 或 TikTok 搜索对应视频教程。
* **自定义与动态随机训练库 (Dynamic Routine Generator):** 支持定义每周训练部位，系统动态随机生成动作组合，对抗肌肉适应性。
* **名人堂与汗水存证 (Hall of Fame & Sweat Evidence):** 用户提交成果，审核通过后由开发者免费将其数据上传至 TON 区块链进行永久存证。
* **全球实时对抗 (LIVE Battles):** 8 种语言社区，提供实时数据看板。
